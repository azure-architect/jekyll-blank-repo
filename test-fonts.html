Paragraph Text

This is a sample paragraph using the Prompt font family. Prompt is a geometric sans-serif typeface designed by Cadson Demak Studio, originally created for Thai text but adapted for Latin scripts. It offers excellent readability at various sizes and weights.

Bold text example: This demonstrates how bold text appears with the Prompt font. The font maintains its clarity and geometric character even at bolder weights.

Small text example: Even at smaller sizes, Prompt maintains excellent legibility and character. This makes it perfect for body text, captions, and detailed information.
